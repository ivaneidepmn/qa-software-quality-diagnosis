# QA Software Quality Diagnosis

Este projeto foi criado para ajudar empresas a identificar problemas de qualidade em seus sistemas através de uma análise estruturada de QA.

## Objetivo

Permitir que empresas descrevam problemas em seus sistemas para que seja possível:

- analisar riscos
- identificar falhas
- criar planos de teste
- propor melhorias de qualidade

## Como funciona

A empresa pode preencher o formulário de diagnóstico de qualidade.

Formulário:

https://docs.google.com/forms/d/e/1FAIpQLSf6slGehypENE1hqV9ftjujWHvlR6O5lc7Tld9LPkCNslfT_Q/viewform

Após o envio, será possível analisar:

- problemas de usabilidade
- falhas funcionais
- gargalos de performance
- riscos de qualidade
- necessidade de automação de testes

## O que posso entregar

✔ Plano de testes  
✔ Estratégia de QA  
✔ Automação de testes  
✔ Relatório de bugs  
✔ Melhoria de experiência do usuário  
✔ Documentação técnica
